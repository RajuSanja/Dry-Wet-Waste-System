# Dry & Wet Waste Segregator

An Arduino-based automated waste segregation system using:
- Moisture Sensor
- Ultrasonic Sensor
- Servo Motor
- Arduino UNO
## Project Demo Video

[Watch Demo Video](https://raw.githubusercontent.com/RajuSanja/Dry-Wet-Waste-System/main/Project_Demo.mp4)

## Features
- Detects wet and dry waste
- Automatic segregation
- Eco-friendly waste management
- Low-cost embedded system

## Technologies Used
- Arduino IDE
- Embedded C
- Sensors & Automation
