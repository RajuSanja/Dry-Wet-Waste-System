# Dry & Wet Waste Segregator

An Arduino-based automated waste segregation system using:
- Moisture Sensor
- Ultrasonic Sensor
- Servo Motor
- Arduino UNO


## Features
- Detects wet and dry waste
- Automatic segregation
- Eco-friendly waste management
- Low-cost embedded system

## Technologies Used
- Arduino IDE
- Embedded C
- Sensors & Automation
